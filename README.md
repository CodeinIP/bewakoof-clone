## Deployment

To deploy this project run

```bash
  vercel --prod
```


## Features

- Add items to cart
- Filtering
- Search items
- Login
- Mock OTP verification

## Run Locally

Clone the project

```bash
  git clone https://github.com/CodeinIP/bewakoof-clone.git
```

Go to the project directory

```bash
  cd bewakoof-clone
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
